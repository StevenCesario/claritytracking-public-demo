ClarityTracking Snippet Test

This page includes the claritytracking-tracker.js snippet.

Check your browser's Developer Console (usually F12) for logs prefixed with [ClarityTracking]. You should see "Tracker Initializing..." followed by "Sending event: PageView..." shortly after the page loads.

Also, check the Network tab in your DevTools for a POST request to /api/ingest/event. Inspect its payload to ensure it contains the correct event_name, event_time, event_id, event_source_url, and the user_data object (including user_agent, fbp/fbc if cookies exist, and any UTM parameters or referrer info).

To test marketing parameters, add some to the URL like:
?utm_source=test_source&utm_medium=test_medium&utm_campaign=test_campaign&source=test_page

Finally, check your backend server logs and database (event_logs table) to confirm the event was received and stored correctly.
